# Java 8 Features 🚀

This repository demonstrates the core features introduced in Java 8 with practical and well-structured examples.

## 📌 Features Covered

- Lambda Expressions
- Functional Interfaces
- Stream API
- Method References
- Optional Class
- Default & Static Methods in Interfaces
- Date and Time API

## 🎯 Objective

The goal of this project is to understand and implement modern Java programming concepts using a functional and declarative approach.

Each example focuses on:
- Writing clean and readable code
- Improving performance using Streams
- Applying functional programming principles

## 🛠 Tech Stack

- Java 8
- JDK 8+
- VS Code / IntelliJ IDEA

## 📂 Project Structure

Each feature is organized in separate classes for better clarity and understanding.
